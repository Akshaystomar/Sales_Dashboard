#Excel_Sales_Dashboard - Project

### Project Objectives  
- Build an interactive sales dashboard using Microsoft Excel.  
- Develop skills in data cleaning, transformations, and visualization with Excel tools (pivot tables, slicers, charts).  
- To learn how to organize raw data and turn it into meaningful visuals.  
- To improve my data analysis and visualization skills.

- ### Tools & skills Used  
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Slicers and Filters  
- Conditional Formatting  
- Data cleaning techniques (removing duplicates, handling missing data)  
- Chart types (bar, line, Pie, etc.)

- ### Insights Gained
- Phones and Chairs were the top-selling categories, generating the highest revenue.
- California had the strongest sales performance among all states.
- Sales peaked in November and December, showing strong year-end growth.
- Technology category showed consistent profit growth from 2021 to 2024, reaching its highest profit in 2024.
